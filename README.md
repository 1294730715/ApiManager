

```
系统特点：

系统支持管理员管理、权限控制、角色管理

支持多项目、多模块划分，同时支持私密项目、模块密码访问

支持数据字典管理、支持数据字典密码访问

支持项目错误码管理、支持接口拷贝等功能

支持前端菜单自定义，支持管理员菜单自定义

支持接口PDF文件导出

接口支持多版本号
```
